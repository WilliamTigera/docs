# About Calico Cloud

## What is {{prodname}}?

{{prodname}} is security solution for cloud-native applications running on containers and Kubernetes. It is the SaaS/pay-as-you-go version of Calico Enterprise, which also includes the core Calico Open Source (Calico CNI networking and network policy).

![calico-cloud](/img/calico-cloud/calico-cloud-features.svg)

Beyond the **Kubernetes security** features that you get from Calico Enterprise and Calico Open Source, {{prodname}} adds these **container security** solutions:

- **Image vulnerability management**

   Image Assurance provides automated image scanning and blocking so you can monitor and assess workloads for new/existing CVEs, 24/7. See [Vulnerability management documentation](/image-assurance/scan-image-registries.mdx).

- **Container threat defense**

    Fully-automated runtime protection against known and unknown attacks (network or container-based). See
    [Container threat defense documentation](/threat/container-threat-detection.mdx).

## Best fit

The sweetspot for {{prodname}} is small teams who need to manage the full spectrum of compliance in a web-based console. To jumpstart learning for teams, {{prodname}} provides:

- Built-in onboarding tutorials in the web console
- Automatic policy recommendations to make it easy for developers to secure their microservices and applications from day one
- Hands-on training from Customer Support during the trial period that targets your use cases  
- Self-service workshops and training to speed up adoption
